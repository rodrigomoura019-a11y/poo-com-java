# Projeto da matéria de "Programação Orientada a Objetos com Java"

## Objetivo: 
**Demonstrar aplicação prática dos conceitos de "Programação Orientada a Objetos"** e **contribuir com reflexões sobre os "Objetivos de Desenvolvimento Sustentável"** por meio do desenvolvimento de um **sistema que simule uma plataforma para conectar doadores, instituições/comunidades e itens  disponíveis para doação**, com  foco em organização, regras de negócio e responsabilidade social. De modo a apoiar o cadastro, gerenciamento e distribuição de itens doados para pessoas ou instituições em situação de  necessidade.

## Detalhamento das funcionalidades: 
- cadastrar doadores;
- cadastrar beneficiários e itens;
- registrar solicitações;
- validar  disponibilidade;
- concluir entregas;
- gerar relatórios simples.

## Tecnologias Utilizadas
- Java openjdk 21.0.4
- Git 2.53.0.windows.2
- Gemini 3.1 Pro
